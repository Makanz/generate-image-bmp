# GenerateImageBmp

Node.js app that generates a dashboard BMP/PNG image (800x480) with system metrics.

## Build

```bash
npm install
```

## Run

```bash
npm run generate
```

Output is saved to `output/dashboard.png`.

## Development

```bash
npm run dev
```

Runs a local web server for the dashboard.

## Test

```bash
npm test
```
